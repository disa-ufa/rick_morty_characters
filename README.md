# Rick & Morty Characters (Flutter)

Мобильное приложение на Flutter, которое загружает список персонажей мультсериала «Рик и Морти» из публичного API, позволяет добавлять персонажей в избранное и поддерживает оффлайн-доступ за счёт локального кеширования.

## Возможности

- **Главный экран (Персонажи)**
  - список персонажей в виде карточек
  - карточка: изображение, имя, статус, вид, локация
  - кнопка ⭐ для добавления/удаления из избранного
  - **пагинация**: подгрузка новых персонажей при скролле

- **Экран “Избранное”**
  - список избранных персонажей (карточки как на главном экране)
  - **сортировка** (по выбранному параметру)
  - удаление из избранного

- **Навигация**
  - `BottomNavigationBar` для переключения между вкладками
  - состояние вкладок сохраняется (скролл/данные не сбрасываются)

- **Оффлайн-режим**
  - загруженные персонажи сохраняются локально (кеш)
  - при отсутствии интернета показываются данные из кеша
  - отображается плашка “Оффлайн: показаны сохранённые данные (кэш)”

- **Тема**
  - светлая/тёмная тема + переключатель

## API

Используется публичный API:
- Документация: https://rickandmortyapi.com/documentation/
- Endpoint: `GET https://rickandmortyapi.com/api/character?page={page}`

## Технологии и зависимости

- State management: `provider` + `ChangeNotifier`
- Сеть: `http`
- Локальное хранилище (кеш персонажей + избранное): `hive`, `hive_flutter`

Полный список зависимостей — в `pubspec.yaml`.
